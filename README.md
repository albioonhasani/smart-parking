# smart-parking
Ky sistem inteligjent parkimi ofron një zgjidhje moderne për menaxhimin dhe përdorimin efikas të hapësirave të parkimit. Përmes një aplikacioni ose platforme online, përdoruesit mund të regjistrohen, të kontrollojnë disponueshmërinë e vendeve të parkimit në kohë reale dhe të rezervojnë një vend sipas nevojës.
